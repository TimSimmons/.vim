# .vim

To get set up run:

```console
cd ~/
git clone --recursive https://github.com/TimSimmons/.vim.git .vim
make install
```

## Git Submodules

To add a submodule:
`git submodule add https://github.com/bling/vim-airline.git bundle/vim-airline`
